Parser implementations and factory for supported NMEA 0183 sentences. Notice that all parsers, excluding the common base class, are visible only within the package scope and not meant to be used directly. Instead, you should use the {@link net.sf.marineapi.nmea.parser.SentenceFactory} and {@link net.sf.marineapi.nmea.io.SentenceReader} classes. Custom parsers can be integrated by extending {@link net.sf.marineapi.nmea.parser.SentenceParser} and registering the new parser in {@link net.sf.marineapi.nmea.parser.SentenceFactory}.
